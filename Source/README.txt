# README.txt

## Dự án: Giúp Ares vượt qua mê cung

### Mục tiêu dự án
Dự án này nhằm phát triển một ứng dụng Python sử dụng các thuật toán tìm kiếm để giúp Ares vượt qua một mê cung phức tạp. Ứng dụng sẽ cho phép Ares di chuyển và đẩy các viên đá lên các công tắc để mở cánh cửa dẫn đến kho báu.

### Cấu trúc thư mục

### Cài đặt môi trường
1. **Cài đặt Python**: Đảm bảo rằng bạn đã cài đặt Python phiên bản 3.6 trở lên.
2. **Cài đặt các thư viện cần thiết**: Chạy lệnh sau để cài đặt các thư viện cần thiết:
pip install -r requirements.txt

### Tài liệu đầu vào
- Các file đầu vào được lưu trữ trong thư mục `Source` và được đặt tên theo định dạng `input-01.txt`, `input-02.txt`, ... với nội dung mô tả mê cung và các viên đá.

### Kết quả đầu ra
- Kết quả sẽ được lưu trong các file đầu ra theo định dạng `output-01.txt`, `output-02.txt`, ... và sẽ bao gồm thông tin về thuật toán, số bước, trọng lượng đá đã đẩy, và các thống kê khác.

### Hướng dẫn sử dụng GUI
1. Mở folder chứa thư mục source.
2. sau đó chạy lệnh pip install -r requirements.txt
3. chạy file MazeVisualize.py, sau đó cửa sổ game sẽ hiện lên
4. các tính năng trong GUI được demo ở video ở link sau: https://youtu.be/-NDkl0KQVKo
